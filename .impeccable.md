## Design Context

### Users

Movie lovers who watch with friends. They're browsing on their phone on the couch or quickly checking the web app to add something to their watchlist. The job: eliminate the "what should we watch?" problem by surfacing instant matches between friends' watchlists and personalized recommendations. Context is casual and social — not a productivity tool.

### Brand Personality

**Warm, social, cinematic.** The interface should feel like a curated cinema lobby, not a spreadsheet. Inviting enough to browse leisurely, focused enough to make decisions quickly.

### Aesthetic Direction

- **Visual tone**: Clean and precise like Linear, but with cinematic warmth — dark backgrounds, golden accents, rich imagery
- **References**: Linear (speed, polish, minimal chrome), but applied to a social/entertainment context
- **Anti-references**: Generic SaaS (no blue gradients, no stock photos), corporate (no stiff layouts, no navy), cluttered social feeds (no information overload), childish/gamified (no cartoon illustrations, no badge systems)
- **Theme**: Dark-first. True black backgrounds (#010101) with golden-yellow primary (#deaf5c) in dark mode, warm brown (#905b20) in light mode
- **Motion**: Purposeful and subtle — ambient glows, gentle fades. Respects `prefers-reduced-motion`

### Design Principles

1. **Content is the hero** — Movie posters, friend avatars, and watchlists are the interface. Chrome should disappear.
2. **Warm precision** — Combine Linear's surgical cleanliness with cinematic warmth. Every pixel is intentional, but nothing feels cold.
3. **Social without noise** — Show friends' activity and matches prominently, but never feel like a feed. Curated, not chaotic.
4. **Fast by default** — Interactions should feel instant. Skeleton states over spinners, optimistic updates, minimal layout shift.
5. **Dark-first, light-compatible** — Design for dark mode as the primary experience (matches the cinematic context), ensure light mode is equally polished.

### Design System Summary

- **Fonts**: Syne (display/headings) + DM Sans (body/UI)
- **Colors**: OKLCH-based, warm gold/brown palette with true black backgrounds
- **Spacing**: 4px base grid
- **Radius**: 10px base, generous rounding (cards at 2xl/18px)
- **Components**: shadcn/ui (web), custom React Native components (mobile)
- **Icons**: lucide-react
- **Borders**: Ultra-subtle (6-8% opacity foreground)
- **Shadows**: Layered, warm-tinted in dark mode

### Accessibility

- WCAG AA compliance (4.5:1 contrast minimum)
- Keyboard navigation support
- Screen reader compatibility
- `prefers-reduced-motion` respected (all animations disabled)
